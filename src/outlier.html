Removing Outliers from a Dataset

Outliers are values that lie significantly outside the normal range of a dataset. Removing outliers is an important step in data cleaning, as they can distort the analysis. One common method of identifying outliers is by using the **Interquartile Range (IQR)**.

Dataset Example

        Original Dataset: [3, 7, 8, 5, 12, 19, 15, 10, 4, 6]

Step 1: Calculate the Five-Number Summary

        Ordered Dataset: [3, 4, 5, 6, 7, 8, 10, 12, 15, 19]

        Minimum = 3
        Q1 (25th percentile) = 4.5 (between 4 and 5)
        Median (Q2) = 7.5 (between 7 and 8)
        Q3 (75th percentile) = 11.5 (between 10 and 12)
        Maximum = 19

Step 2: Calculate the Interquartile Range (IQR)

        IQR = Q3 - Q1 = 11.5 - 4.5 = 7

Step 3: Determine the Lower and Upper Boundaries for Outliers

        Lower Bound = Q1 - 1.5 * IQR = 4.5 - 1.5 * 7 = 4.5 - 10.5 = -6
        Upper Bound = Q3 + 1.5 * IQR = 11.5 + 1.5 * 7 = 11.5 + 10.5 = 22

Step 4: Identify and Remove Outliers

        Any data point less than -6 or greater than 22 is considered an outlier.
        In this dataset:
        - No value is less than -6.
        - The maximum value is 19, which is within the range of -6 to 22.
        
        Therefore, there are no outliers to remove in this case.

Step 5: (Hypothetical) If There Were Outliers

If there were outliers beyond the calculated boundaries, you could remove them. For example, if the dataset had a value like 25, it would be considered an outlier and would be removed.

        After removing outliers (if any), the updated dataset would look like:
        [3, 4, 5, 6, 7, 8, 10, 12, 15]
