Univariate and Bivariate Analysis Plots

Numeric features

  • The prefix ‘Uni’ means one, meaning ‘univariate analysis’ is the analysis of one variable at a time.
  • For numeric features, we want to know the range of values present and how often these values (or groups of values) occur.
  • For categorical features, we want to know the number of unique classes and how frequently they occur.

Histogram

A histogram groups values into ranges (or bins), and the height of a bar shows how many values fall in that range.

Box Plot

A box plot summarizes the distribution of a dataset, showing the median, quartiles, and possible outliers.

Density Plot

A density plot shows the distribution of a variable, smoothed by a kernel density estimate.

Violin Plot

The violin plot features a combination of a box plot and a kernel density plot. This means that in addition to showing the quartiles, it also lays out the underlying distribution such as the presence and location of different peaks.

Line Plot

A line plot shows the trend of a variable over time or another continuous variable.

Categorical features

  • These are columns with a limited number of possible values. Examples are sex, country, or age group.
  • Before creating the plots, we’ll first run the summary statistics that show information such as the number of unique classes per feature.
  • This will inform us which features can be effectively visualized. If there are too many classes, the plots are cluttered and unreadable.

Count Plot

A count plot compares different classes of a categorical feature and how often they occur. Think of a bar chart with the bar height showing the number of times each class occurs in the data.

Pie Chart

A pie chart displays the percentage distribution of a categorical variable in a circular graph.

Bivariate Analysis

Bivariate analysis typically examines relationships between two variables, often illustrated with scatter plots or correlation matrices.
